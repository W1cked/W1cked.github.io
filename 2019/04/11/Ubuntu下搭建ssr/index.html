ubuntu中搭建ssr

安装

更新软件源

sudo apt-get update

安装pip(管理python工具)安装docker也需要用到。

sudo apt-get install python-pip
sudo apt-get install python-m2crypto

通过pip安装shadowsocks

sudo pip install shadowsocks

安装过程中可能会出现警告推荐用
sudo -H pip install shadowsocks

配置shadowsocks

在/home路径下建一个shadowsocks.conf文本文件

sudo gedit shadowsocks.conf

添加信息

配置和win上几乎一样，配过的应该会填

配置完成后运行

sudo sslocal -c /home/w1cked/shadowsocks.conf -d start启动
sudo sslocal -c /home/shadowsocks.conf -d stop终止

配置浏览器

可以直接用火狐自带浏览器，需要一个更改代理的插件，win上并不需要，原来一般都是直接上的。

插件应该也很熟悉，配合burpsuit的Proxy SwitchyOmega在代理服务器中的代理协议选择 socks5，本地代理服务器 127.0.0.1，代理窗口1080。
这样就可以在Ubuntu下使用ssr了
